# 001 - Ubuntu Clean Install
**Date:** April 6, 2026
**Project:** MacBook Pro 2015 → Ubuntu Linux Conversion
**Phase Completed:** Phase 1 & 2 — Backup & Clean Install

---

Today I converted my 2015 MacBook Pro from macOS to Ubuntu 24.04 LTS. The decision was driven by Apple no longer providing security updates for this hardware. The machine was still physically sound, so rather than letting good hardware go to waste, I chose to repurpose it with a modern, fully supported operating system.

Before beginning the install, I backed up what little data remained on the drive, as I had already made a habit of keeping minimal files on this machine. With the backup complete, I downloaded Balena Etcher to prepare a bootable flash drive and downloaded the Ubuntu ISO for installation.

The process was not without challenges. The first two Ubuntu downloads were corrupted, causing Balena Etcher to fail during the flash process. On the third attempt the download completed cleanly and the installation proceeded without issue. Total time invested was approximately seven hours, including troubleshooting and breaks.

This project served two purposes. First, extending the life of functional hardware by replacing an unsupported operating system with a modern Linux environment. Second, developing my ability to work with AI as a technical guide — learning how to prompt effectively, interpret results, and course correct when things went wrong.
