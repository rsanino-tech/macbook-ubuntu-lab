# System Overview

This system was converted from a MacBook Pro (2015) into a dedicated Ubuntu-based lab environment for learning Linux, networking, and security concepts.

## Build Summary

The system was rebuilt through a structured process that included:

- Performing a clean installation of Ubuntu 24.04 LTS
- Configuring the system post-install for stability and usability
- Applying basic security hardening practices
- Setting up a local environment for experimentation and learning
- Optimizing system performance for older hardware
- Customizing the terminal and workflow for daily use
- Documenting each step for reproducibility

Detailed steps for this process can be found in:

linux-setup-phases/

## Purpose

This machine serves as the primary lab system for:

- Linux command line practice
- File system management
- Remote access (SSH)
- Network scanning and mapping
- Future security and cloud experiments
